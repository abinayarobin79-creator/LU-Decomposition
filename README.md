# LU Decomposition 

## AIM:
To write a program to find the LU Decomposition of a matrix.

## Equipments Required:
1. Hardware – PCs
2. Anaconda – Python 3.7 Installation / Moodle-Code Runner

## Algorithm
1. Import the NumPy library and the lu function from the scipy.linalg module.
2. Read the order and elements of the square matrix from the user.
3. Use the lu() function to decompose the matrix into Lower triangular matrix (L) and Upper triangular matrix (U).
4. Display the L and U matrices.
